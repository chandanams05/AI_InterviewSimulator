# AI Interview Simulator

An AI-powered mock interview platform where users can:
- Register and log in
- Select their target job role
- Attend role-specific mock interviews
- Get real-time answer evaluation using Gemini API
- Store interview data and questions in MongoDB

## Tech Stack
- Backend: Python / Flask
- Frontend: HTML, CSS, JavaScript
- Database: MongoDB
- AI Evaluation: Gemini API
